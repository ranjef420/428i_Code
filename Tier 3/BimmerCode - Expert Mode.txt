FA Codes (VO Coding)
───────────────────────────────────────────────────────────

VO1) Activate Active High Beam Assistant (Anti-Dazzle)
Purpose: Enable automatic high beam control without dazzling other drivers.
Warning: Requires vehicle equipped with High Beam Assistant. May also require Variable Light Distribution enabled.
Coding Type: VO
ECU/Module: FEM_BODY and FLA/KAFAS2 (depending on equipment)
Prerequisites: Remove 5AP from vehicle FA; perform VO code.
iDrive Impact: Enables system to move high beams independently.
Values: N/A
Verify: High beam automatically adapts without blinding.
Rollback/Notes: CAFD modifications available for HBA speed thresholds (see below).

CAFD modifications REQUIRED after VO coding:
- Speed HBA Enables: FEM_BODY – C_HBA_ENA_V_HI (HEX value for kph)
- Speed HBA Disables: FEM_BODY – C_HBA_ENA_V_LO (HEX value for kph)
- Highway Mode Entry: FEM_BODY – C_HBA_HGWY_V_HI (Default: 8C = 140 kph)
- Highway Mode Exit: FEM_BODY – C_HBA_HGWY_V_LO (Default: 5A = 90 kph)

VO2) Activate Variable Light Distribution
Purpose: Adapt low/high beam distribution based on speed.
Warning: Requires FLA/KAFAS camera installed.
Coding Type: VO
ECU/Module: FEM_BODY
Prerequisites: Remove 8S4 from vehicle FA; perform VO code.
iDrive Impact: City mode: road illuminated closely, highway: lamps raise.
Values: N/A
Verify: Light beam adapts as speed changes.
Rollback/Notes: May be required for full Active High Beam Assistant functionality.

VO3) Activate Enhanced Bluetooth
Purpose: Enable Bluetooth audio streaming, album art, and USB music streaming.
Warning: Requires HU_NBT or combox with HU_CIC/HU_ENTRY.
Coding Type: VO
ECU/Module: CMB_MEDIA and HU_CIC/HU_ENTRY or HU_NBT
Prerequisites: Change 6NH to 6NK (no BMW Assist) or 6NS (with BMW Assist) in FA; perform VO code.
iDrive Impact: Adds BT audio streaming and enhanced features.
Values: N/A
Verify: Pair device, stream music, check album art.
Rollback/Notes: Revert FA codes to default for rollback.

VO4) Activate iDrive Office
Purpose: Enable Office menu in iDrive (contacts, email, SMS receiving).
Warning: Sometimes disabled by Country Controllers.
Coding Type: VO
ECU/Module: CMB_MEDIA and HU_CIC/HU_ENTRY or HU_NBT
Prerequisites: Remove Country Controller code (SL8Ax) from FA; perform VO code.
iDrive Impact: Office menu appears in iDrive.
Values: N/A
Verify: Office menu options available.
Rollback/Notes: Revert Country Controller code if needed.

VO5) Activate Sport Automatic Transmission (SAT)
Purpose: Enable faster/smoother shifts and Sport+ if not already active.
Warning: Standalone BMW option; paddle shifters optional.
Coding Type: VO
ECU/Module: EGS, ICM, KOMBI
Prerequisites: Change 205 to 2TB in FA; perform VO code.
iDrive Impact: Transmission and Sport+ features enabled.
Values: N/A
Verify: Sport+ available, shifting behavior changed.
Rollback/Notes: Return FA to original code for rollback.

HEX Information
Most values (werte) are in HEX. Convert to decimal for true value.
Examples: 08 = 8 ; 1F = 31 ; 32 = 50 ; 64 = 100 ; FF = 255
Units: Speed = km/h ; Time = seconds/hours ; Distance = meters

Highlight References
Blue Highlight = Enabled/Working on my vehicle
Yellow Highlight = Check Section Description
Red Highlight = Issue
