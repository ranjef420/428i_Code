(*** VO CODING ***)

Activate Active High Beam Assistant (Anti-Dazzle)
- Requires the vehicle to be equipped with High Beam Assistant.
- Enables the system to move high beams independently without blinding other drivers.
- MODIFY THE READ FA: Remove 5AP from the vehicle FA.
- VO CODE: FEM_BODY and FLA/KAFAS2 (depending on equipment).

CAFD modifications of HBA functionality after VO coding in FEM_BODY and FLA/KAFAS2:
- Speed HBA Enables: FEM_BODY – C_HBA_ENA_V_HI (value in HEX for kph)
- Speed HBA Disables: FEM_BODY – C_HBA_ENA_V_LO (value in HEX for kph)
- Speed to immediately enter Highway Mode: FEM_BODY – C_HBA_HGWY_V_HI (Default: Werte = 8C = 140 kph)
- Speed to leave Highway Mode: FEM_BODY – C_HBA_HGWY_V_LO (Default: Werte = 5A = 90 kph)

Note: It may be required to enable Variable Light Distribution for Active High Beam Assistant to function fully.

Activate Variable Light Distribution
- Requires an FLA/KAFAS camera installed in the vehicle.
- Adapts light beam based on vehicle speed.
- City: low beam to illuminate close road without dazzling oncoming drivers.
- Highway: lamps raise to increase visibility.
- MODIFY THE READ FA: Remove 8S4 from the vehicle FA.
- VO CODE: FEM_BODY.

Activate Enhanced Bluetooth
- Requires HU_NBT or combox with HU_CIC or HU_ENTRY.
- Enables Bluetooth audio streaming, artwork, and USB music streaming.
- MODIFY THE READ FA: Change 6NH to 6NK (without BMW Assist) or 6NS (with BMW Assist).
- VO CODE: CMB_MEDIA and HU_CIC/HU_ENTRY or HU_NBT.

Activate iDrive Office
- Sometimes disabled by Country Controllers.
- Enables Office Menu in iDrive (Contacts, Email, Text Message Receiving, etc.).
- MODIFY THE READ FA: Remove Country Controller code (SL8Ax) from vehicle FA.
- VO CODE: CMB_MEDIA and HU_CIC/HU_ENTRY or HU_NBT.

Activate Sport Automatic Transmission (SAT)
- Standalone option offered by BMW (paddle shifters optional).
- Enables faster, smoother shifts with ZF8HP Transmission.
- Enables Sport+ if not already active.
- MODIFY THE READ FA: Change 205 to 2TB in the vehicle FA.
- VO CODE: EGS, ICM, and KOMBI.

(*** VO CODING ***)

CAFD Coding Instructions (FDL Coding)
───────────────────────────────────────────────────────────
It’s recommended to VO code first before doing any CAFD coding, 
since VO coding will return values to their defaults.

Highlight References
- Blue Highlight = Enabled/Working on my vehicle
- Yellow Highlight = Check Section Description
- Red Highlight = Issue

HEX Information
- Most values (werte) will be in HEX. Convert to Decimal for true value.
  Examples: 08 = 8 ; 1F = 31 ; 32 = 50 ; 64 = 100 ; FF = 255
- Units: Speed = km/h ; Time = seconds/hours ; Distance = meters

Angel LED Brightness (Xenon Headlamps or LED Headlamps required)
- Adjusts the brightness of the Angel LEDs.
- Values are in HEX. Example:
  0% = 00, 25% = 19, 50% = 32, 100% = 64
- Note: Supposedly does not work on F32.

Cornering Lights (Fog Lamps) & Fog Lamp Welcome Lights
- Fog lights illuminate in the steered direction.
- Front fog lights are required for all features below.

LED Replacement Turn Signal Bulbs Activation

- Traditional turn signal bulbs are checked every 4 minutes by sending a small pulse. 
- With LEDs, this pulse causes flicker. The following codes remove pulse monitoring 
  and enable LED bulb recognition.
- Recommended bulb: CREE 30W XB-D5 LED.
───────────────────────────────────────────────────────────

───────────────────────────────────────────────────────────
Ineffective or Unavailable Codes
───────────────────────────────────────────────────────────
- The following codes have been tested and determined to be ineffective 
  or unavailable on F30/F32 builds.  
- Some only apply to other chassis, older modules, or non-U.S. builds.

HU_CIC - 6NR (BMW Apps) cannot be coded on CIC head units 
  if not originally installed (requires FSC code).

HU_ENTRY - Video in Motion not available.  
  Head unit hardware limitation.

FEM_BODY - Adaptive LED Headlights cannot be coded if not factory-installed.  
  Requires hardware (headlamps, modules, sensors).

KOMBI - HUD retrofits not available by coding alone.  
  Requires hardware HUD windshield and projector.

CMB_MEDIA - Combox functions not available if vehicle was never 
  equipped with combox hardware (applies to early F30 builds).
